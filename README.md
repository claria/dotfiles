# dotfiles


Repository of my private dotfiles.

```
.dotfiles
│   dotman.py
│   README.md
│
│   dotfile1
│   dotfile2
└───folder1
│   └───dotfile3
│   └───dotfile4
└───hosts
    └───machine1
    │   └───dotfile3
    └───machine2
        └───dotfile3
```

## dotman.py

Dotfile management script.

> dotman.py install

Test123

> dotman.py add /path/to/.dotfile1 /path/to/.dotfile2
